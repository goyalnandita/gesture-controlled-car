# Gesture Controlled Car using Arduino

## Description
A gesture-controlled car built using Arduino UNO that moves based on hand movements.

## Working
Hand gestures are captured using sensors and transmitted wirelessly to control the car’s direction and movement in real time.

## Features
- Gesture-based navigation  
- Wireless control  
- Real-time response  

## Technologies Used
- Arduino UNO  
- Embedded C  
- Sensors  
- Wireless Module
